# Lab 1 - Prosty kalkulator w Javie

## Opis projektu
Program w Javie, który pyta o dwie liczby, dodaje je i pokazuje wynik.

## Jak uruchomić
1. Skompiluj kod:  
   `javac src\Main.java`

2. Uruchom program:  
   `java -cp src Main`

3. Podaj dwie liczby, zobacz wynik.
